About MobileBoil

MobileBoil starts where Skeleton Boilerplate left off, adding a new breakpoint of 1440px for a wider and greater visual experience, and implementing a mobile-first approach.

Three Core Principles

  • A Responsive Grid Up To Large Desktops: Elegant scaling from mobile to tablets to browsers.
  • Fast to Start: It's a tool for rapid development with best practices
  • Style Agnostic: It provides the most basic, beautiful styles, but is meant to be overwritten.

Docs & Support

MobileBoil is open-source and has a project on git. Check it out if you want to report bugs or create a pull request.
